Ordenar un proyecto.

Crear 3 archivos: un archivo HTML, un archivo CSS y una imagen (svg). Crear una carpeta general que será la raíz del proyecto, donde habrá subcarpetas para ordenar cada uno de los archivos antes mencionados. Recordá que quizás —y en general encontrarán esta estructura— uno de ellos no necesita estar dentro de una subcarpeta y puede permanecer directo en la raíz.
